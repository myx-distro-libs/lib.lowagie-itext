# lib.lowagie-itext

AE3 packaging wrapper for **iText** — a Java PDF library.

Provides: `ae3.pkg.lib.itext`

## Bundled library

- **iText 2.1.7** by Bruno Lowagie and Paulo Soares (Lowagie BVBA)
  - Homepage: https://itextpdf.com
  - License: GNU Lesser General Public License v2 (LGPLv2)
  - Note: iText 2.x is LGPLv2. iText 5 and later are AGPL — this package uses the 2.x branch only.
- **Bouncy Castle Provider 1.43** (`bcprov-jdk16-143.jar`) — cryptography support for iText PDF encryption
  - Homepage: https://www.bouncycastle.org
  - License: MIT License

This repository contains only pre-built jar files and AE3 build metadata.
No source code from the above libraries is included or modified.
